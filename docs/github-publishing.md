# GitHub Publishing Notes

Before publishing:

1. Check syntax:
   - `php -l` for PHP files.
   - `node --check public/assets/js/chatbot.js`.
2. Search for private paths and secrets.
3. Confirm `.gitignore` excludes `.env`, dumps, archives, logs, and private reports.
4. Replace placeholder GitHub links in `marketing/linkedin-post.md`.
5. Add repository topics:
   - `chatbot`
   - `php`
   - `mysql`
   - `arabic`
   - `erp`
   - `ollama`
   - `local-ai`
   - `no-api-key`

Suggested repo description:

`Free local PHP/MySQL chatbot for websites, ERP, reports, and databases. Arabic/English, no API tokens required, optional Ollama.`
