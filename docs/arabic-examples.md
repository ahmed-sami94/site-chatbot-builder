# Arabic Examples

Use these scenarios during setup and QA:

- `اعرض تقرير المبيعات الشهر ده`
- `قارن بين المنتجين دول`
- `فين بيانات الفاتورة رقم 123؟`
- `ابحث عن منتج بالكود A123`
- `اعرض المنتجات الأقل مبيعًا`
- `لم أجد الإجابة، حولني لمستشار`

Expected behavior:

- Reply in Arabic.
- Convert Arabic digits.
- Use allowed database/report sources only.
- Provide source labels.
- Refuse private data and write actions.
- Ask for narrowing details when confidence is low.
